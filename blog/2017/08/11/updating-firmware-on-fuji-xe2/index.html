Updating Firmware on Fuji XE2

I always forget how to do this, so I wrote some notes.

Erase Memory Card:
Camera on > Menu OK > Left > down at the bottom

See existing firmware:
Hold Disp/Back button and Switch Camera On

Go to fuji site download latest firmware on computer,
put file on memory card via computer.

Battery full charged > put memory card in camera.

Hold Disp/Back button and Switch Camera On again.

Should see ‘Firmware Upgrade’ > OK > OK

WAIT!!! Don’t do anything with camera whilst it’s updating.

Should see confirmation when it’s done.

Turn off, voila!

McrFRED 01 Videos

Videos taken from the first McrFRED.

Thanks to Chris at Three Quarters Productions for filming and editing.

My first go at a presentation, any feedback would be great :bowtie:.

In the presentation I cover the FRED movement, a bit about my background, some things I remember from when I first started building websites and how things have changed.

Then getting a bit more technical, showing some code examples, how I configure some apps and various tools I enjoy to use in my workflow to be more productive.

The videos are quite long, so, if you're only interested in a particular section, I've added links so you can jump straight to that section.

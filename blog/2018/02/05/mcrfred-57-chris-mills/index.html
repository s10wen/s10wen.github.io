McrFRED 57 Chris Mills

For the next McrFRED, Chris Mills from Mozilla will be returning. This time he'll be talking about "Doing things better: A more efficient, usable web?".

Meetup Page

Chris Mills - @chrisdavidmills

Title

Doing things better: A more efficient, usable web?

Description

In recent years, a number of features have appeared on the web platform that allow us to provide better user experiences, largely through doing things more efficiently rather than completely new patterns. In this talk, Mozilla’s Chris Mills explores a few of these features — such as Streams, Service workers and PWAs — and why they are worth knowing about as well more towards the future.

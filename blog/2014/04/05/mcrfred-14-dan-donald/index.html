McrFRED 14 Dan Donald

Manchester FRED fourteen is a go! With the one and only Dan Donald talking about responsive web design.

The event will take place on Thursday 29th May, 6.30pm.

At the new location TechHub Manchester v2.0 on Lever Street.

Thanks to everyone that’s filled the feedback form so far, if you haven’t please do, it really helps!

https://s10wen.wufoo.com/forms/mcrfred-feedback/

You asked and I deliver! From the feedback, it seems responsive design is still a topic that needs some attention.

So here we have it, Dan Donald @hereinthehive on responsive design.

We’ve worked with responsive design for a few years and explored what we can do with the flexibility it provides but what happens when we start to explore ‘element queries’?

When modules within a site can look at both the browser and it’s parent element to make layout choices, that opens up a new level of curiosity and complexity.

There’s many ways of making this work now, but is this something we want and can we see browsers implementing it?

Tickets are limited to 40 and will go quick, so please sign up early to avoid disappointment.

Tickets have all gone, but as usual I’ve set up a reserve list in case anyone pulls out or can’t make it.

Thanks, look forward to seeing you then.

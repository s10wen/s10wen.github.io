McrFRED 15 Simon Owen

Manchester FRED fifteen. For this one I'm going to be going over a recent project covering the project from start to finish.

The event will take place on Thursday 26th June, 6.30pm.

At the new location TechHub Manchester v2.0 on Lever Street.

These days Front-End Developers have a lot to think about, in this talk I aim to get through as many of them as possible. There's millions of ways to go about things, I'll be going over some of the things I take into consideration and techniques I use to help me along the way.

Some of the things I'll be covering:

  • Yeoman
  • Bower
  • Grunt / Gulp
  • Sass
  • Cli
  • SVGs
  • Responsive Design
  • Performance Testing
  • Jank Busting
  • Device Testing
  • Browser Testing
  • Workflow Techniques
  • Shortcuts, shortcuts, shortcuts
  • Optimisation
  • Accessibility
  • Build Scripts
  • Local Development Environment
  • Dotfiles

Tickets are limited to 50 go grab one from here: http://mcrfred15.eventbrite.co.uk

Thanks, look forward to seeing you then.

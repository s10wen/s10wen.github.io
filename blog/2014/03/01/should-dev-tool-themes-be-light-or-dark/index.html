Should dev tool themes be light or dark?

I spend hours every day writing code and find that having a dark theme is a lot easier on the eye.

Using high contrast colours such as pink, green, yellow and blue I find it’s much easier to skim read and get to what I’m after fast.

So much so, I wrote a dark theme for Chrome Dev Tools based on my favourite theme - Monokai.

http://devthemez.com/themes/so-dark-monokai-v3

Currently I’m working with Maurice making Chrome Dev Theming easier using LESS.

https://github.com/mauricecruz/zero-base-themes/blob/master/themes/s10wen-monokai.less

Also Firefox have done a great job on their dark theme:

Firefox > dev tools > settings cog > DevTools theme: Dark = Great job @firefox pic.twitter.com/qv7nw1DpA3

— Simon Owen (@s10wen) February 6, 2014

I’m interested to know what you preference is, light or dark?
